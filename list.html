There are 3 types of list

definition list
which shows terms and definitions
unordered list
which shows bullets
ordered list
which show numbers or letters

Above is an example of a definiton list.

unordered list can have a type of attribute which can be applied to the UL or the LI tag.
The three values allowed are:

  • type = 'square'
  • type = 'disc'
  • type = 'circle'

Ordered list can have a type attribute too.
Used along with a start attribute they can show:

  1. lowercase letters
  2. uppercase letters
  3. lower case roman numerals
  4. upper case roman numbers
  5. numbers
